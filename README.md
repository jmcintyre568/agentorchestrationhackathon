HACK YOUR FUTURE
by Jasper, Sahib, and Ari

Features:

 - An AST resume scanner that looks for holes in your resume that will cause it to be trashed before it ever reaches a recruiter

 - An orchestrator agent that assigns tasks to specialized subagents that scrape the internet for information on the employee using OSINT and corporate intel, verifies it with a council of agents that determine if the information is true, then merges the information with a "judge" agent using Weights&Biases Weave

 - A predictive "Event Radar" that analyzes corporate footprints to forecast which upcoming industry events the interviewer is likely to attend for real-world networking 

 - An automated cross-referencing engine that maps shared interests between candidates and interviewers to generate custom "cold start" icebreaker questions as well as good questions/follow-ups to mention during an interview
